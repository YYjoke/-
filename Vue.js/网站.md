git六部曲：

1.创建文件夹

2.创建可管理仓库   $ git init

3.把文件加如仓库（等待区）   $ git add readme.txt

4.把文件加如仓库   $ git commit -m "wrote a readme file" //引号内为提交说明 内容任意 不可没有

5.关联远程仓库  $ git remote add origin git@github.com:yyjoke/learngit.git

6.远程推送  $ git push -u origin master
